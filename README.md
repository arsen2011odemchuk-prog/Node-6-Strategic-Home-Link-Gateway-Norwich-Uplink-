# 🏠 Node 6: Strategic Home-Link Gateway (Norwich Uplink)

**Node 6** is the stationary counterpart to the Node 5 tactical terminal. Located at the primary compute site in Norwich, it is responsible for maintaining the 160km (100-mile) point-to-point radio beam to London. It acts as the high-speed "Uplink" for the Node 4 AI cluster.

## 🚀 Key Strategic Features
*   **160km Symmetrical Link:** Paired with Node 5, it creates a focused, pencil-thin radio corridor using a 25dBi parabolic dish.
*   **Window-Optimized Feed:** Designed with a high-gain offset to minimize signal reflection and loss when transmitting through dual-pane glass.
*   **Integrated UPS (Uninterruptible Power):** Features a LiFePO4 battery backup, allowing the link to remain active for 48 hours during local power outages.
*   **Hardware Bridge Logic:** Uses a dedicated network processor to handle AES-256 encryption overhead, ensuring zero-latency commands for the RTX 3060 AI server.

## 🛠 Operation & Safety
Node 6 operates as a transparent bridge. It automatically reconnects to Node 5 using a proprietary handshake protocol. In the event of a security breach at the home site, Node 6 can be remotely deactivated from the London terminal, physically severing the radio link to protect the Node 4 AI cluster.
